## Overview

This guide will show you how to set up different types of AI agent workflows with Trigger.dev. The examples take inspiration from Athropic's blog post on [building effective agents](https://www.anthropic.com/research/building-effective-agents).

| Pattern | Example |
|:--------|:--------|
| Prompt chaining | [Generate and translate copy](/guides/ai-agents/generate-translate-copy) |
| Routing | [Route a user question to a different model depending on its complexity](/guides/ai-agents/route-question) |
| Parallelization | [Respond to customer while checking if their text is inappropriate](/guides/ai-agents/respond-and-check-content) |
| Orchestrator | [Check if a news article is factually correct](/guides/ai-agents/verify-news-article) |
| Evaluator-optimizer | [Translate text, evaluate if good, feedback and repeat if not](/guides/ai-agents/translate-and-refine) |
