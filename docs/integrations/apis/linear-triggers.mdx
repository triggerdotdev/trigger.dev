## All Triggers

### `onAttachment`
    
When any action is performed on an attachment. [Official Linear docs](https://developers.linear.app/docs/graphql/attachments).

```ts example.ts
client.defineJob({
  id: "job-example-attachment",
  name: "Linear Example Attachment",
  version: "0.1.0",
  integrations: {
    linear,
  },
  // this should trigger when creating an attachment in the job above
  trigger: linear.onAttachment(),
  run: async (payload, io, ctx) {
    await io.linear.createAttachment("create-attachment", {
      issueId: payload.issueId,
      url: "https://example.com"
    });
  }
});
```
